Política de Privacidade - Cloud Cost Watcher

Última atualização: Janeiro de 2026

Bem-vindo ao Cloud Cost Watcher! Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos suas informações pessoais ao usar nosso aplicativo móvel. Ao usar o Cloud Cost Watcher, você concorda com as práticas descritas aqui.

1. Informações que Coletamos

1.1 Dados de Autenticação

  • Conta Google: Quando você faz login via Google Sign-In, coletamos seu nome, endereço de e-mail e ID único da conta Google, conforme fornecido pelo Firebase Authentication.
  • Dados de Uso: Informações sobre como você interage com o app, como custos de nuvem monitorados (AWS, Azure, GCP), mas não armazenamos dados financeiros sensíveis diretamente.

1.2 Dados Técnicos

  • Dispositivo: Modelo do dispositivo, sistema operacional, versão do app e identificadores únicos (como ID do dispositivo).
  • Logs de Uso: Dados sobre sessões, erros e desempenho do app, coletados automaticamente pelo Firebase.

1.3 Dados Voluntários

Qualquer informação adicional que você fornecer ao configurar provedores de nuvem (como credenciais de API), que são armazenadas localmente no dispositivo ou criptografadas no Firestore.

2. Como Usamos Suas Informações

  • Autenticação e Segurança: Para verificar sua identidade e proteger sua conta.
  • Funcionalidade do App: Para monitorar e exibir custos de nuvem de provedores suportados (AWS, Azure, GCP).
  • Melhoria do Serviço: Para analisar uso, corrigir bugs e melhorar recursos.
  • Comunicações: Para enviar notificações sobre custos ou atualizações do app (com seu consentimento).

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins comerciais.

3. Compartilhamento de Informações

  • Provedores de Serviços: Compartilhamos dados com o Firebase (Google) para autenticação e armazenamento, e com provedores de nuvem (AWS, Azure, GCP) apenas para consultar custos, conforme suas permissões.
  • Requisitos Legais: Podemos divulgar dados se exigido por lei ou para proteger direitos legais.
  • Transferências Internacionais: Dados podem ser processados em servidores do Google (EUA), sujeitos às leis locais.

4. Segurança dos Dados

Implementamos medidas de segurança padrão da indústria, incluindo criptografia (TLS/SSL), autenticação Firebase e armazenamento seguro no Firestore. No entanto, nenhum método de transmissão ou armazenamento eletrônico é 100% seguro.

5. Seus Direitos

De acordo com a LGPD (Lei Geral de Proteção de Dados do Brasil), você tem o direito de:

  • Acessar, corrigir ou excluir seus dados pessoais.
  • Revogar consentimento para processamento.
  • Solicitar portabilidade de dados.

Para exercer esses direitos, entre em contato conosco pelo e-mail abaixo.

6. Retenção de Dados

Mantemos seus dados apenas pelo tempo necessário para os fins descritos. Dados de autenticação são retidos enquanto sua conta estiver ativa; logs técnicos são excluídos após 90 dias.

7. Alterações a Esta Política

Podemos atualizar esta política periodicamente. Notificaremos sobre mudanças significativas via e-mail ou notificação no app. O uso contínuo do app após alterações constitui aceitação.
